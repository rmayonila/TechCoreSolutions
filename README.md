# TechCore Solutions - Employee & Payroll Management System

## Project Overview

A complete ASP.NET Core MVC CRUD application for managing employee information and payroll transactions. Built with C# and Entity Framework Core.

## Features

### Employee Module

- ✅ Add new employees
- ✅ View all employees
- ✅ Edit employee information
- ✅ Delete employees
- ✅ View payroll history per employee

### Payroll Module

- ✅ Add payroll transactions
- ✅ View payroll history per employee
- ✅ Edit payroll records
- ✅ Delete payroll records
- ✅ Automatic calculation of Gross Pay and Net Pay

### Business Logic

- Gross Pay = Days Worked × Daily Rate
- Net Pay = Gross Pay - Deduction
- Input validation (no negative values)
- Referential integrity enforcement

## Technology Stack

- **Framework:** ASP.NET Core MVC (.NET 10)
- **Language:** C# only
- **ORM:** Entity Framework Core 9.0
- **Database:** SQL Server (LocalDB/Express) or MySQL (XAMPP phpMyAdmin)
- **UI:** Razor Views with Bootstrap CSS

## Database Setup

### Option 1: Using MySQL (XAMPP)

1. **Start XAMPP:**
   - Open XAMPP Control Panel
   - Start Apache and MySQL services
   - Open phpMyAdmin at http://localhost/phpmyadmin

2. **Create Database:**
   - In phpMyAdmin, create a new database named `TechCoreSolutionsDB`

3. **Update Connection String:**
   - The default connection string in `appsettings.Development.json` is:
     ```
     Server=localhost;Port=3306;Database=TechCoreSolutionsDB;Uid=root;Pwd=;
     ```
   - If MySQL uses a password, update: `Pwd=your_password;`

4. **Run Migrations:**
   ```bash
   dotnet ef database update
   ```

### Option 2: Using SQL Server

1. **Install SQL Server Express or LocalDB:**
   - Download from: https://www.microsoft.com/en-us/sql-server/sql-server-downloads

2. **Update Connection String in `appsettings.json`:**

   ```
   Server=(localdb)\mssqllocaldb;Database=TechCoreSolutionsDB;Trusted_Connection=true;TrustServerCertificate=true;
   ```

   Or for SQL Server Express:

   ```
   Server=.\SQLEXPRESS;Database=TechCoreSolutionsDB;Trusted_Connection=true;TrustServerCertificate=true;
   ```

3. **Run Migrations:**
   ```bash
   dotnet ef database update
   ```

## Running the Application

### Prerequisites

- .NET SDK 10.0 or later installed
- Either MySQL (XAMPP) or SQL Server running

### Steps

1. **Restore Dependencies:**

   ```bash
   dotnet restore
   ```

2. **Build the Project:**

   ```bash
   dotnet build
   ```

3. **Update Database:**

   ```bash
   dotnet ef database update --context ApplicationDbContext
   ```

4. **Run the Application:**

   ```bash
   dotnet run
   ```

5. **Access the Application:**
   - Open browser to: https://localhost:7156 (or port shown in console)
   - The app will redirect to the Employee Management page

## Key Features Explained

### Employee Model

- **EmployeeID** (Auto-generated Primary Key)
- **FirstName, LastName** (Required, up to 50 chars)
- **Position** (Required, up to 100 chars)
- **Department** (Required, up to 100 chars)
- **DailyRate** (Required, decimal with 2 decimal places)
- **Payrolls** (Navigation property for related payroll records)

### Payroll Model

- **PayrollID** (Auto-generated Primary Key)
- **EmployeeID** (Foreign Key)
- **Date** (Transaction date)
- **DaysWorked** (0-31 days)
- **Deduction** (Amount to deduct from gross pay)
- **GrossPay** (Computed: DaysWorked × Employee.DailyRate)
- **NetPay** (Computed: GrossPay - Deduction)
- **Employee** (Navigation property)

## Validation Rules

### Employee

- All fields are required
- Names limited to 50 characters
- Position/Department limited to 100 characters
- Daily Rate must be greater than 0

### Payroll

- Employee must be selected
- Date is required
- Days Worked must be 0-31
- Deduction cannot be negative
- All values validated on form and server-side

## Database Schema

### Employees Table

```sql
CREATE TABLE Employees (
    EmployeeID INT PRIMARY KEY IDENTITY(1,1),
    FirstName NVARCHAR(50) NOT NULL,
    LastName NVARCHAR(50) NOT NULL,
    Position NVARCHAR(100) NOT NULL,
    Department NVARCHAR(100) NOT NULL,
    DailyRate DECIMAL(18,2) NOT NULL
);
```

### Payrolls Table

```sql
CREATE TABLE Payrolls (
    PayrollID INT PRIMARY KEY IDENTITY(1,1),
    EmployeeID INT NOT NULL,
    Date DATETIME NOT NULL,
    DaysWorked INT NOT NULL,
    Deduction DECIMAL(18,2) NOT NULL,
    FOREIGN KEY (EmployeeID) REFERENCES Employees(EmployeeID) ON DELETE CASCADE
);
```

## Troubleshooting

### Database Connection Issues

- **MySQL not found:** Make sure XAMPP is running and MySQL service is started
- **SQL Server not found:** Ensure SQL Server Express or LocalDB is installed
- **Connection timeout:** Check firewall settings and database server status

### Migration Issues

- **"Unable to create DbContext":** Ensure database server is running
- **"Pending changes":** Run `dotnet ef migrations add <MigrationName>` first

### Port Already in Use

- Application runs on https://localhost:7156 by default
- If port is busy, modify in `Properties/launchSettings.json`
